# 📖 Ситуация

В компании **PrintFlow** используется настольное приложение для регистрации и обработки заданий на печать документов.

Приложение позволяет сотрудникам печатного центра и операторам:

* создавать новые задания на цветную, черно-белую или черновую печать;
* отслеживать статус выполнения заданий;
* управлять жизненным циклом заданий печати через табличный интерфейс.

В корпоративной инфраструктуре приложение может запускаться:

* через удалённые рабочие столы (RDP);
* внутри защищённых виртуальных сред (VDI);
* через терминальные серверы.

В таких условиях традиционная автоматизация интерфейса на основе **UI-селекторов** может быть недоступна или нестабильна.

Поэтому используется подход **Machine Vision**, при котором робот:

* анализирует изображение интерфейса;
* находит элементы по их визуальному шаблону;
* выполняет действия по координатам найденных элементов.

Дополнительно в данной работе используется **интеграция с искусственным интеллектом (LLM)**.
Робот будет **генерировать наименования документов для печати с помощью GPT**, а затем автоматически добавлять их в систему.

В рамках практической работы необходимо разработать **RPA-процесс в Puzzle RPA**, который:

* взаимодействует с приложением через **Machine Vision**;
* использует **GPT для генерации данных**.

# Форма регистрации задания

Форма используется для создания нового задания на печать.

Пользователь вводит данные и добавляет запись в таблицу.

---

## Поле Document

Назначение:

ввод наименования или описания документа.

В данной работе значение **генерируется автоматически с помощью GPT**.

---

## Поле Operator

Назначение:

имя оператора, оформившего задание.

Примеры:

* Ivan Petrov
* Anna Smirnova

---

## Список Print Type

Назначение:

выбор типа печати.

Возможные значения:

* Color
* BlackWhite
* Draft

---

## Список Priority

Назначение:

выбор приоритета выполнения задания.

Возможные значения:

* Low
* Medium
* High

## Что необходимо сделать
