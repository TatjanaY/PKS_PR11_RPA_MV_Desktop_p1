# 📖 Ситуация

В компании **ReturnFlow** используется настольное приложение для регистрации и обработки заявок на возврат товаров.

Приложение позволяет сотрудникам отдела поддержки и менеджерам:

* создавать новые заявки на возврат по различным причинам;
* отслеживать статус их рассмотрения;
* управлять жизненным циклом возвратов через табличный интерфейс.

В корпоративной инфраструктуре приложение может запускаться:

* через удалённые рабочие столы (RDP);
* внутри защищённых виртуальных сред (VDI);
* через терминальные серверы.

В таких условиях традиционная автоматизация интерфейса на основе **UI-селекторов** может быть недоступна или нестабильна.

Поэтому используется подход **Machine Vision**, при котором робот:

* анализирует изображение интерфейса;
* находит элементы по их визуальному шаблону;
* выполняет действия по координатам найденных элементов.

Дополнительно в данной работе используется **интеграция с искусственным интеллектом (LLM)**.
Робот будет **генерировать описания причин возврата с помощью GPT**, а затем автоматически добавлять их в систему.

В рамках практической работы необходимо разработать **RPA-процесс в Puzzle RPA**, который:

* взаимодействует с приложением через **Machine Vision**;
* использует **GPT для генерации данных**.

В данной работе значение **генерируется автоматически с помощью GPT**.

* Ivan Petrov
* Anna Smirnova

# Шаг 2. Генерация описаний причин возврата через GPT

## Зачем выполняется шаг

Робот должен **самостоятельно придумать описания причин возврата**, используя искусственный интеллект.

Это демонстрирует интеграцию **LLM в RPA-процесс**.

---

## Что необходимо сделать

Использовать блок:

**AI → Выполнить запрос в GPT**

### Параметры блока

| Параметр                   | Значение                                                                                                                                                                                                                                                                                             |
| -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Выполнить запрос к         | PuzzleGPT                                                                                                                                                                                                                                                                                            |
| Запрос                     | Сгенерируй 10 кратких описаний причин возврата товаров для интернет-магазина. Распредели по категориям: 4 Defect (брак, неисправность, повреждение при доставке), 3 Size (не подошёл размер, ошибка в размерной сетке), 3 Other (не понравился цвет, передумал, не соответствует описанию). Верни только список описаний без номеров, каждое описание с новой строки. |
| Максимальное число токенов | 500                                                                                                                                                                                                                                                                                                  |
| Уровень случайности        | 0.5                                                                                                                                                                                                                                                                                                  |

---

## Пример ответа GPT

```
Обнаружен брак корпуса при распаковке
Не работает кнопка включения после первого использования
Повреждена упаковка и товар при доставке курьером
Не подошёл размер обуви по длине стопы
Ошибка в размерной сетке на сайте
Маленький размер одежды относительно заявленного
Не понравился фактический цвет товара
Передумал и решил отказаться от покупки
Товар не соответствует описанию на странице
Несоответствие комплектации заявленной на сайте
```
