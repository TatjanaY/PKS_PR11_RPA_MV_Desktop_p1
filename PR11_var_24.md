# 📖 Ситуация

В учебном заведении **EduTrack** используется настольное приложение для регистрации и учёта посещаемости студентов.

Приложение позволяет преподавателям и методистам:

* создавать новые записи о посещении лекций, практических занятий и экзаменов;
* отслеживать статус отметки посещаемости;
* управлять жизненным циклом записей через табличный интерфейс.

В корпоративной инфраструктуре приложение может запускаться:

* через удалённые рабочие столы (RDP);
* внутри защищённых виртуальных сред (VDI);
* через терминальные серверы.

В таких условиях традиционная автоматизация интерфейса на основе **UI-селекторов** может быть недоступна или нестабильна.

Поэтому используется подход **Machine Vision**, при котором робот:

* анализирует изображение интерфейса;
* находит элементы по их визуальному шаблону;
* выполняет действия по координатам найденных элементов.

Дополнительно в данной работе используется **интеграция с искусственным интеллектом (LLM)**.
Робот будет **генерировать имена студентов для тестовых записей с помощью GPT**, а затем автоматически добавлять их в систему.

В рамках практической работы необходимо разработать **RPA-процесс в Puzzle RPA**, который:

* взаимодействует с приложением через **Machine Vision**;
* использует **GPT для генерации данных**.

# Форма регистрации записи

Форма используется для создания новой записи о посещении занятия.

Пользователь вводит данные и добавляет запись в таблицу.

---

## Поле Student

Назначение:

ввод имени студента.

В данной работе значение **генерируется автоматически с помощью GPT**.

---

## Поле Teacher

Назначение:

имя преподавателя, ведущего занятие.

Примеры:

* Ivan Petrov
* Anna Smirnova

---

## Список Lesson Type

Назначение:

выбор типа занятия.

Возможные значения:

* Lecture
* Practice
* Exam

---

## Список Priority

Назначение:

выбор приоритета обработки записи.

Возможные значения:

* Low
* Medium
* High

# Шаг 2. Генерация имён студентов через GPT

## Зачем выполняется шаг

Робот должен **самостоятельно сгенерировать тестовые имена студентов**, используя искусственный интеллект.

Это демонстрирует интеграцию **LLM в RPA-процесс**.

---

## Что необходимо сделать

Использовать блок:

**AI → Выполнить запрос в GPT**

### Параметры блока

| Параметр                   | Значение                                                                                                                                                                                                                                                                                             |
| -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Выполнить запрос к         | PuzzleGPT                                                                                                                                                                                                                                                                                            |
| Запрос                     | Сгенерируй 10 вымышленных имён студентов (фамилия и имя) для учебного заведения EduTrack. Используй разнообразные российские имена. Верни только список имён без номеров, каждое имя с новой строки. |
| Максимальное число токенов | 500                                                                                                                                                                                                                                                                                                  |
| Уровень случайности        | 0.5                                                                                                                                                                                                                                                                                                  |
