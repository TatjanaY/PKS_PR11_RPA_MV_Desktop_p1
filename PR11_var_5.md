# 📖 Ситуация

В компании **ClientFlow** используется настольное приложение для регистрации и обработки данных новых клиентов.

Приложение позволяет менеджерам и сотрудникам отдела продаж:

* создавать новые профили клиентов (розничных, корпоративных, VIP);
* отслеживать статус их активации;
* управлять жизненным циклом клиентов через табличный интерфейс.

В корпоративной инфраструктуре приложение может запускаться:

* через удалённые рабочие столы (RDP);
* внутри защищённых виртуальных сред (VDI);
* через терминальные серверы.

В таких условиях традиционная автоматизация интерфейса на основе **UI-селекторов** может быть недоступна или нестабильна.

Поэтому используется подход **Machine Vision**, при котором робот:

* анализирует изображение интерфейса;
* находит элементы по их визуальному шаблону;
* выполняет действия по координатам найденных элементов.

Дополнительно в данной работе используется **интеграция с искусственным интеллектом (LLM)**.
Робот будет **генерировать наименования клиентов с помощью GPT**, а затем автоматически добавлять их в систему.

В рамках практической работы необходимо разработать **RPA-процесс в Puzzle RPA**, который:

* взаимодействует с приложением через **Machine Vision**;
* использует **GPT для генерации данных**.

Форма используется для создания новой записи клиента.

Пользователь вводит данные и добавляет запись в таблицу.

## Поле Client Name

Назначение:

ввод наименования клиента или компании.

В данной работе значение **генерируется автоматически с помощью GPT**.

---

## Поле Manager

Назначение:

имя менеджера, ответственного за клиента.

Примеры:

* Ivan Petrov
* Anna Smirnova

---

## Список Segment

Назначение:

выбор клиентского сегмента.

Возможные значения:

* Retail
* Business
* VIP

## Список Priority

Назначение:

выбор приоритета обработки.

Возможные значения:

* Low
* Medium
* High

# Шаг 2. Генерация наименований клиентов через GPT

## Зачем выполняется шаг

Робот должен **самостоятельно придумать наименования клиентов/компаний** для разных сегментов, используя искусственный интеллект.

Это демонстрирует интеграцию **LLM в RPA-процесс**.

---

## Что необходимо сделать

Использовать блок:

**AI → Выполнить запрос в GPT**

### Параметры блока

| Параметр                   | Значение                                                                                                                                                                                                                                                                                             |
| -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Выполнить запрос к         | PuzzleGPT                                                                                                                                                                                                                                                                                            |
| Запрос                     | Сгенерируй 10 наименований клиентов или компаний для CRM-системы. Распредели по сегментам: 4 Retail (розничные покупатели, локальные магазины), 3 Business (B2B партнёры, поставщики, сервисные компании), 3 VIP (премиум клиенты, крупные инвесторы, эксклюзивные клиенты). Верни только список названий без номеров, каждое название с новой строки. |
| Максимальное число токенов | 500                                                                                                                                                                                                                                                                                                  |
| Уровень случайности        | 0.5                                                                                                                                                                                                                                                                                                  |

---
