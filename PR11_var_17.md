# 📖 Ситуация

В учебной платформе **CourseFlow** используется настольное приложение для регистрации и обработки записей на онлайн-курсы.

Приложение позволяет сотрудникам отдела обучения и методистам:

* создавать новые записи на курсы для начинающих, среднего и продвинутого уровней;
* отслеживать статус их запуска;
* управлять жизненным циклом курсов через табличный интерфейс.

В корпоративной инфраструктуре приложение может запускаться:

* через удалённые рабочие столы (RDP);
* внутри защищённых виртуальных сред (VDI);
* через терминальные серверы.

В таких условиях традиционная автоматизация интерфейса на основе **UI-селекторов** может быть недоступна или нестабильна.

Поэтому используется подход **Machine Vision**, при котором робот:

* анализирует изображение интерфейса;
* находит элементы по их визуальному шаблону;
* выполняет действия по координатам найденных элементов.

Дополнительно в данной работе используется **интеграция с искусственным интеллектом (LLM)**.
Робот будет **генерировать названия онлайн-курсов с помощью GPT**, а затем автоматически добавлять их в систему.

В рамках практической работы необходимо разработать **RPA-процесс в Puzzle RPA**, который:

* взаимодействует с приложением через **Machine Vision**;
* использует **GPT для генерации данных**.

# Шаг 2. Генерация названий курсов через GPT

## Зачем выполняется шаг

Робот должен **самостоятельно придумать названия онлайн-курсов** для разных уровней сложности, используя искусственный интеллект.

Это демонстрирует интеграцию **LLM в RPA-процесс**.

---

## Что необходимо сделать

Использовать блок:

**AI → Выполнить запрос в GPT**

### Параметры блока

| Параметр                   | Значение                                                                                                                                                                                                                                                                                             |
| -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Выполнить запрос к         | PuzzleGPT                                                                                                                                                                                                                                                                                            |
| Запрос                     | Сгенерируй 10 названий онлайн-курсов для учебной платформы. Распредели по уровням: 4 Beginner (введение в программирование, основы веб-разработки, начало работы с данными, дизайн для новичков), 3 Intermediate (анализ данных, автоматизация тестирования, управление проектами), 3 Advanced (машинное обучение, архитектура облачных систем, кибербезопасность). Верни только список названий без номеров, каждое название с новой строки. |
| Максимальное число токенов | 500                                                                                                                                                                                                                                                                                                  |
| Уровень случайности        | 0.5                                                                                                                                                                                                                                                                                                  |
