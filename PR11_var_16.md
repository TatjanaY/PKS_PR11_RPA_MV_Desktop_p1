# 📖 Ситуация

В компании **DeviceFlow** используется настольное приложение для регистрации и учёта корпоративного оборудования.

Приложение позволяет специалистам IT-отдела и сотрудникам склада:

* создавать новые записи о ноутбуках, принтерах, серверах и других устройствах;
* отслеживать статус их регистрации в корпоративной системе;
* управлять жизненным циклом записей через табличный интерфейс.

В корпоративной инфраструктуре приложение может запускаться:

* через удалённые рабочие столы (RDP);
* внутри защищённых виртуальных сред (VDI);
* через терминальные серверы.

В таких условиях традиционная автоматизация интерфейса на основе **UI-селекторов** может быть недоступна или нестабильна.

Поэтому используется подход **Machine Vision**, при котором робот:

* анализирует изображение интерфейса;
* находит элементы по их визуальному шаблону;
* выполняет действия по координатам найденных элементов.

Дополнительно в данной работе используется **интеграция с искусственным интеллектом (LLM)**.
Робот будет **генерировать наименования устройств с помощью GPT**, а затем автоматически добавлять их в систему.

В рамках практической работы необходимо разработать **RPA-процесс в Puzzle RPA**, который:

* взаимодействует с приложением через **Machine Vision**;
* использует **GPT для генерации данных**.

Форма используется для создания новой учётной записи оборудования.

Пользователь вводит данные и добавляет запись в таблицу.

ввод наименования или модели устройства.

В данной работе значение **генерируется автоматически с помощью GPT**.

---

* Ivan Petrov
* Anna Smirnova

## Список Device Type

* Laptop

* Server

## Список Priority

Назначение:

Возможные значения:

* Low

# Шаг 2. Генерация наименований устройств через GPT

## Зачем выполняется шаг

Робот должен **самостоятельно придумать названия устройств** для разных типов оборудования, используя искусственный интеллект.

Это демонстрирует интеграцию **LLM в RPA-процесс**.

---

## Что необходимо сделать

Использовать блок:

**AI → Выполнить запрос в GPT**

### Параметры блока

| Параметр                   | Значение                                                                                                                                                                                                                                                                                             |
| -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Выполнить запрос к         | PuzzleGPT                                                                                                                                                                                                                                                                                            |
| Запрос                     | Сгенерируй 10 наименований корпоративных устройств для учёта. Распредели по типам: 4 Laptop (ноутбуки для сотрудников, мобильные рабочие станции), 3 Printer (сетевые МФУ, лазерные принтеры), 3 Server (стоечные серверы, системы хранения). Верни только список названий без номеров, каждое название с новой строки. |
| Максимальное число токенов | 500                                                                                                                                                                                                                                                                                                  |
| Уровень случайности        | 0.5                                                                                                                                                                                                                                                                                                  |

---

# Шаг 3. Преобразование ответа GPT в массив
