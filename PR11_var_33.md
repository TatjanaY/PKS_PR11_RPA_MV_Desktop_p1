# 📖 Ситуация

В компании **RentFlow** используется настольное приложение для регистрации и обработки заявок на аренду недвижимости.

Приложение позволяет менеджерам и агентам:

* создавать новые заявки на аренду (квартиры, офисы, коммерческие помещения);
* отслеживать статус их подтверждения;
* управлять жизненным циклом аренд через табличный интерфейс.

В корпоративной инфраструктуре приложение может запускаться:

* через удалённые рабочие столы (RDP);
* внутри защищённых виртуальных сред (VDI);
* через терминальные серверы.

В таких условиях традиционная автоматизация интерфейса на основе **UI-селекторов** может быть недоступна или нестабильна.

Поэтому используется подход **Machine Vision**, при котором робот:

* анализирует изображение интерфейса;
* находит элементы по их визуальному шаблону;
* выполняет действия по координатам найденных элементов.

Дополнительно в данной работе используется **интеграция с искусственным интеллектом (LLM)**.
Робот будет **генерировать наименования объектов недвижимости с помощью GPT**, а затем автоматически добавлять их в систему.

В рамках практической работы необходимо разработать **RPA-процесс в Puzzle RPA**, который:

* взаимодействует с приложением через **Machine Vision**;
* использует **GPT для генерации данных**.

# Форма регистрации аренды

Форма используется для создания новой заявки на объект недвижимости.

Пользователь вводит данные и добавляет запись в таблицу.

---

## Поле Property

Назначение:

ввод наименования или описания объекта недвижимости.

В данной работе значение **генерируется автоматически с помощью GPT**.

---

## Поле Manager

Назначение:

имя менеджера, ответственного за аренду.

Примеры:

* Ivan Petrov
* Anna Smirnova

---

## Список Rental Type

Назначение:

выбор типа аренды.

Возможные значения:

* Short-term
* Long-term
* Commercial

---

## Список Priority

Назначение:

выбор приоритета обработки заявки.

Возможные значения:

* Low
* Medium
* High

---

# Шаг 2. Генерация наименований объектов через GPT

## Зачем выполняется шаг

Робот должен **самостоятельно придумать названия объектов недвижимости**, используя искусственный интеллект.

Это демонстрирует интеграцию **LLM в RPA-процесс**.

---

## Что необходимо сделать

Использовать блок:

**AI → Выполнить запрос в GPT**

### Параметры блока

| Параметр                   | Значение                                                                                                                                                                                                                                                                                             |
| -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Выполнить запрос к         | PuzzleGPT                                                                                                                                                                                                                                                                                            |
| Запрос                     | Сгенерируй 10 наименований объектов недвижимости для системы RentFlow. Распредели по типам: 4 Short-term (апартаменты, квартиры посуточно), 3 Long-term (дома, квартиры на длительный срок), 3 Commercial (офисы, склады, торговые площади). Верни только список названий без номеров, каждое название с новой строки. |
| Максимальное число токенов | 500                                                                                                                                                                                                                                                                                                  |
| Уровень случайности        | 0.5                                                                                                                                                                                                                                                                                                  |

---

## Пример ответа GPT

```
Апартаменты с видом на парк

Двухэтажный коттедж в пригороде
Офис класса А на проспекте
Складское помещение 200 м²
Торговое помещение в ТЦ
Квартира с евроремонтом
Лофт для творческой мастерской
Особняк с садом
```

# Шаг 3. Преобразование ответа GPT в массив
