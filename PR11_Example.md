# 📖 Ситуация

В компании **TechFlow** используется настольное приложение для регистрации и обработки сервисных заявок сотрудников.

Приложение позволяет сотрудникам:

* создавать новые заявки;
* отслеживать их статус;
* управлять обработкой заявок через таблицу.

В корпоративной инфраструктуре приложение может запускаться:

* через удалённые рабочие столы;
* внутри виртуальных рабочих сред;
* через терминальные серверы.

В таких условиях традиционная автоматизация интерфейса на основе **UI-селекторов** может быть недоступна или нестабильна.

Поэтому используется подход **Machine Vision**, при котором робот:

* анализирует изображение интерфейса;
* находит элементы по их изображению;
* выполняет действия по координатам найденных элементов.

Дополнительно в данной работе используется **интеграция с искусственным интеллектом (LLM)**.
Робот будет **генерировать названия заявок с помощью GPT**, а затем автоматически добавлять их в систему.

В рамках практической работы необходимо разработать **RPA-процесс в Puzzle RPA**, который:

* взаимодействует с приложением через **Machine Vision**;
* использует **GPT для генерации данных**.

# Шаг 2. Генерация названий заявок через GPT

## Зачем выполняется шаг

Робот должен **самостоятельно придумать названия заявок**, используя искусственный интеллект.

Это демонстрирует интеграцию **LLM в RPA-процесс**.

---

## Что необходимо сделать

Использовать блок:

**AI → Выполнить запрос в GPT**

### Параметры блока

| Параметр                   | Значение                                                                                                                                                                                                                                                                                             |
| -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Выполнить запрос к         | PuzzleGPT                                                                                                                                                                                                                                                                                            |
| Запрос                     | Сгенерируй 10 названий заявок для IT-отдела. Темы: доступ к CRM, настройка рабочей станции, проблемы с почтой, VPN, установка ПО, настройка принтера, сетевые проблемы, сброс пароля, обновление ПО, доступ к базе данных. Верни только список названий без номеров, каждое название с новой строки. |
| Максимальное число токенов | 500                                                                                                                                                                                                                                                                                                  |
| Уровень случайности        | 0.5                                                                                                                                                                                                                                                                                                  |

---

## Пример ответа GPT

```
Настройка VPN для удалённого сотрудника
Сброс пароля в учётной записи
Обновление антивирусных баз
Доступ к CRM для нового сотрудника

Настройка корпоративной почты
Подключение сетевого принтера
```
