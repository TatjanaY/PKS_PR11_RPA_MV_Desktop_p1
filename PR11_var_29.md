# 📖 Ситуация

В компании **ContentFlow** используется настольное приложение для регистрации и обработки материалов для публикации (новости, статьи блога, отчеты и другие типы контента).

Приложение позволяет сотрудникам редакционного отдела и контент-менеджерам:

* создавать новые записи о публикациях;
* отслеживать статус их готовности к публикации;
* управлять жизненным циклом материалов через табличный интерфейс.

В корпоративной инфраструктуре приложение может запускаться:

* через удалённые рабочие столы (RDP);
* внутри защищённых виртуальных сред (VDI);
* через терминальные серверы.

В таких условиях традиционная автоматизация интерфейса на основе **UI-селекторов** может быть недоступна или нестабильна.

Поэтому используется подход **Machine Vision**, при котором робот:

* анализирует изображение интерфейса;
* находит элементы по их визуальному шаблону;
* выполняет действия по координатам найденных элементов.

Дополнительно в данной работе используется **интеграция с искусственным интеллектом (LLM)**.
Робот будет **генерировать заголовки статей с помощью GPT**, а затем автоматически добавлять их в систему.

В рамках практической работы необходимо разработать **RPA-процесс в Puzzle RPA**, который:

* взаимодействует с приложением через **Machine Vision**;
* использует **GPT для генерации данных**.

# Шаг 2. Генерация заголовков статей через GPT

## Зачем выполняется шаг

Робот должен **самостоятельно придумать названия статей** для разных типов контента, используя искусственный интеллект.

Это демонстрирует интеграцию **LLM в RPA-процесс**.

---

## Что необходимо сделать

Использовать блок:

**AI → Выполнить запрос в GPT**

### Параметры блока

| Параметр                   | Значение                                                                                                                                                                                                                                                                                             |
| -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Выполнить запрос к         | PuzzleGPT                                                                                                                                                                                                                                                                                            |
| Запрос                     | Сгенерируй 10 заголовков статей для контент-системы ContentFlow. Распредели по типам: 4 News (релизы продуктов, обновления безопасности, анонсы), 3 Blog (тренды технологий, кейсы использования, заметки разработчиков), 3 Report (финансовые отчеты, аналитика рынка, итоги квартала). Верни только список заголовков без номеров, каждый заголовок с новой строки. |
| Максимальное число токенов | 500                                                                                                                                                                                                                                                                                                  |
| Уровень случайности        | 0.5                                                                                                                                                                                                                                                                                                  |

---
