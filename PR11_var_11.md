# 📖 Ситуация

В компании **DeliveryFlow** используется настольное приложение для регистрации и обработки логистических доставок.

Приложение позволяет диспетчерам и менеджерам склада:

* создавать новые заявки на доставку посылок;
* отслеживать статус их выполнения;
* управлять жизненным циклом доставок через табличный интерфейс.

В корпоративной инфраструктуре приложение может запускаться:

* через удалённые рабочие столы (RDP);
* внутри защищённых виртуальных сред (VDI);
* через терминальные серверы.

В таких условиях традиционная автоматизация интерфейса на основе **UI-селекторов** может быть недоступна или нестабильна.

Поэтому используется подход **Machine Vision**, при котором робот:

* анализирует изображение интерфейса;
* находит элементы по их визуальному шаблону;
* выполняет действия по координатам найденных элементов.

Дополнительно в данной работе используется **интеграция с искусственным интеллектом (LLM)**.
Робот будет **генерировать наименования посылок с помощью GPT**, а затем автоматически добавлять их в систему.

В рамках практической работы необходимо разработать **RPA-процесс в Puzzle RPA**, который:

* взаимодействует с приложением через **Machine Vision**;
* использует **GPT для генерации данных**.

Форма используется для создания новой заявки на доставку.

Пользователь вводит данные и добавляет запись в таблицу.

## Поле Package

Назначение:

ввод наименования или описания посылки.

В данной работе значение **генерируется автоматически с помощью GPT**.

---

Назначение:

* Ivan Petrov

Назначение:

выбор типа доставки.

Возможные значения:

* Standard

## Список Priority

Назначение:

выбор приоритета обработки доставки.

Возможные значения:

* Low
* Medium
* High

# Шаг 2. Генерация наименований посылок через GPT

## Зачем выполняется шаг

Робот должен **самостоятельно придумать описания посылок** для разных типов доставки, используя искусственный интеллект.

Это демонстрирует интеграцию **LLM в RPA-процесс**.

---

## Что необходимо сделать

Использовать блок:

**AI → Выполнить запрос в GPT**

### Параметры блока

| Параметр                   | Значение                                                                                                                                                                                                                                                                                             |
| -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Выполнить запрос к         | PuzzleGPT                                                                                                                                                                                                                                                                                            |
| Запрос                     | Сгенерируй 10 наименований посылок для логистической службы. Распредели по типам: 4 Standard (документы, офисные принадлежности, одежда), 3 Express (электроника, медицинские препараты), 3 Same-day (продукты, срочные контракты). Верни только список названий без номеров, каждое название с новой строки. |
| Максимальное число токенов | 500                                                                                                                                                                                                                                                                                                  |
| Уровень случайности        | 0.5                                                                                                                                                                                                                                                                                                  |

---

## Пример ответа GPT

```
Пакет бухгалтерской отчётности

Зимняя куртка (размер M)
Серверный блок питания
Аптечка первой помощи
Срочный договор поставки

```

# Шаг 3. Преобразование ответа GPT в массив
